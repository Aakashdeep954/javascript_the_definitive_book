4.13.4 The delete operator

delete is a unary operator that attempts to object property or an array element specified as its operand.when a property is deleted, the property ceases to exist. Attempting to read a non existent property returns undefined.Deleting an array element leaves a hole but does not change the array length.

4.13.5 The await operator

Await was introduced in 2017 as a way to make asynchronous programming more natural in javascript.Await expects a promise object as operand.It does not block other asynchronous operations from proceding at the same time.
