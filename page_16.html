find() and findIndex()

The find() and findIndex() are similar to filter in that they iterate through the array looking for the element for which your predicate function returns a truthy value.Unlike filter() these two methods stop iterating the first time the predicate finds the an element.If no matching element is found find() returns undefined and findIndex() returns -1.

every() and some()

The every() method is like the mathematical for all quantifier:it returns true if and only if your predicate function returns true for all elements in the array. The some() method is like the mathematical there exists quatifier:it returns true if there exists atleast one element in the array for which your predicate function return true and returns true if predicate function returns false for all the elments in of array.

reduce() and reduceRight()

The reduce() and reduceRight() combine the elements of an array using the function you specify to produce a single value. reduce() takes two arguments.First is the function that performs the reduction.The second argument is an initial value to pass to the function.ReduceRight() works just like the reduce() except it processes the array from highest index to lowest index.

7.8.2 Flattening arrays with flat() and flatMap()

In ES2019 the flat() flat method creates and returns a new array that contains the same elements as the array it is called on, except that any elements that are themeselves arrays are flattened into a returned array.

7.8.3 Adding arrays with concat()

The concat() method creates and returns a new array that contains the elements of original array on which the concat() was invoked,follwed by each of arguments to concat().
