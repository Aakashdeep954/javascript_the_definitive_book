javascript programs are written using unicode character set and you can use any unicode charater in strings and comments. The language also allows use of unicode digits , letters, ideographs but not emojis in identifiers.

2.5.1

some computer hardware and software cannot output or process the full set of unicode characters. To support these systems javascript defines escape sequences that allow us to write unicode characters using ASCII characters. The uicode escapes begin with characters \u either followed by exactly four hexadecimal digits or one to six hexadecimal digits in curly brackets.
