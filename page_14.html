CHAPTER 7

Arrays



7.1 Creating arrays

There are several ways to create an array.They are

  • Array literal
  • The spread operator
  • The array constructor
  • The array.of() and array.from() factory methods
7.1.1 Array literals

By far the simplest way to create an array is with an array literal, which is simply comma seperated list of array elements within the square brackets.

7.1.2 Spread operator

In ES6 and later you can use spread operator to include the elements of one array within an array literal.The three dots spread the elements of an array into an array literal that is being created.The spread operator works on any iterable object.

7.1.3 Array constructor

Another method to create an array is with the Array constructor.You can invoke this constructor in three distinct ways.

  • call it with no arguments
  • call it with single argument which specifies the lenght
  • Explicitly specify two or more elements or a single non numeric element
