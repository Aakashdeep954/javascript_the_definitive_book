3.10.3 Destructuring assignment

ES6 implements a kind of compound declaration and assignment syntax known as destructuring assignment. In destructuring assignment the right hand side of the equal sign is an object or array and the left hand specifies one more variables using a syntax that mimics array or object literal syntax. The number of variables on the left side of the desturcturing assignment does have to match the number of variables on the right side. Extra variables on the left are set to undefined and on the right are ignored.
