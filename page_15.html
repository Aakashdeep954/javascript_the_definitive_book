7.1.5 Array.of()

Array constructor cannot be used to create with single argument.In ES6, the Array.of() function addresses this problem. It is a factory method that creates and returns a new array irrespective of number of arguments.

7.1.6 Array.from()

Array.from() is another factory method that was introduced in ES6.It expects an iterable or array like object as its first argument and returns a new array that contains the elements of that object.

7.3 sparse arrays

A sparse array is one in which the elements do not have continous indexes.If the array is sparse the value of the lenght property is more than the number or elements.

7.8.1 Array iterator methods

forEach

The forEach method ieterates through an array invoking the function you specified for each element.forEach invokes the funcition with three arguments array value,index,array itself.If you only care about the array value just pass argument.

map()

The map function passes each element of the array it was invoked on to the function you specify and returns an array containing values returned by your function.

filter()

The filter method returns an array containing a subset of elements of the array on which it is invoked. The function you pass should be a predicate: a function that returns true or false. If the return value is true or a value that converts to true,then the element passed to the predicate is member of the subset and is added to the array that will become the return value.
