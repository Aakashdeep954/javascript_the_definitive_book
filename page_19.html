CHAPTER 8

Functions



8.1 Defining functions

The most straightforward way to define a javascript function is with the function keyword which can be used as declaration and expression. ES6 defines new way to define function without using function keyword known as arrow function.

8.1.1 Function declaration

Function declaration statement are hoisted to the top of the enclosing script,function or block so that the functions defined in this way may be invoked from code that appears before the definition.

8.1.3 Arrow functions

In ES6 you can define functions using a particular compact syntax known as arrow function.If the body of the arrow function is a single statement you can omit the return keyword and the curly braces.If the arrow function has single argument you can omit the parenthesis.

8.2 Invoking functions

functions can be invoked in five ways.

  • As functions
  • As methods
  • As constuctors
  • Idirectly through their call() and bind()
  • Implicitly,via javascript langauge features that do not appear like normal function invocations.
