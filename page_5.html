3.8 Immutable primitive values and mutable object references

There is fundamental difference between primitive values and objects. Primitives are immutable whereas objects are mutable.primitives are compared by value.Objects are not compared by value. Two objects are equal even if they have same properites and values.Two distinct arrays are equal if they have same elements in same order. TWo objects are same if and only if they refer to same underlying object.

3.9 Type conversions

Javascript is very flexible about type conversions.If javascript wants a string it will convert whatever the value you give it to a string.If javascript wants a number it will convert whatever the value you give it to a number or to NaN if it cannot perform a meaningful conversion.
